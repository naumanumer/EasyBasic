About

This project is the continuity of GwBasic.Net.

Name Convention

GwBasic.Net was named as 'GwBasic.Net' due to its production using .NET technologies i.e. C# (winform). But as soon as the technology changes form C# to Electron.js, the name, logo and design also changed. And the new name (EasyBasic) and technology also, was suggested by Adnan Umer.

GwBasic.Net

Before, I started working on EasyBasic its about 3 attempts of building the GwBasic.Net with charming and much user friendly even on Windows XP. But, unfortunately i was not abled to do that as I want. These 3 attempts as also avaible on Github.

EasyBasic

I also made several minor and 2 major attempts using Electron.js, one of which this version is.

Features

Here are some of the feautures included in the this build:

  • Compeletly customized GUI
  • Supports Multi-tasking
  • Show Recent File
  • Built-in PcBasic
  • File Comparer
  • Complete GwBasic Documentation with real time search
  • About 60 built-in Examples
  • Six different Themes
  • IntelliSense
  • Bracket matching
  • Multiple Selections
  • Column (box) text selection
  • Folding
