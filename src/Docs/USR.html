USR

                
                    value = USR[n](expr)

Returns zero.

Parameters

  • n is a digit [0—9].
  • expr is an expression.

Notes

  • In GW-BASIC, calls a machine-code function and returns its return value.
  • This function is not implemented in PC-BASIC.

Errors

  • n is not a digit [0—9]: Syntax error.
