    CLS

    Syntax¶

    1
    CLS [region]

    region has numeric value that describes the region to clear:

    • 0 clears both text and graphics
    • 1 clears graphics only
    • 2 clears text only

    Note

    Any thing after region upto and of statement (:) or line or file is ignored
